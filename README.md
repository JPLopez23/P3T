# Proyecto 3 – Teoría de la Computación

Este proyecto implementa una **Máquina de Turing** para encriptar y desencriptar mensajes mediante el **Cifrado César**. La simulación respeta las operaciones básicas de una Máquina de Turing: cambio de estado, sustitución de símbolos en la cinta y movimiento del cabezal.

## Instalación

1. **Clona el repositorio**: `git clone https://github.com/JPLopez23/P3T.git`
2. **Navega al directorio**: `cd P3T`
3. **Ejecuta el programa**: `python TC.py`

## Estructura del Proyecto

### Archivos principales:
- **`TC.py`**: Implementación de la Máquina de Turing y menú interactivo
- **`casos_encriptar.txt`**: Casos de prueba para encriptación
- **`casos_decriptar.txt`**: Casos de prueba para desencriptación
- **`maquina_turing_especificacion.json`**: Especificación formal de la Máquina de Turing
